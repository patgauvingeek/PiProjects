install dependencies
--------------------
sudo apt-get install git sqlite3 libsqlite3-dev wiringpi expat libexpat1 libexpat1-dev libssl-dev [sqlite3-doc]

install litesql
---------------
0.3.17.0 (0.3.18.0 has a bug: https://sourceforge.net/p/litesql/bugs/48)

make
sudo make install
